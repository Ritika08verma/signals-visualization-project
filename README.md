# Signals and Systems Visualization GUI

This project is a Python-based GUI that helps visualize basic signals and their transformations.

## Features
- Sine, Cosine, Step, Ramp, Exponential, Impulse signals
- Amplitude scaling & shifting
- Time shifting (advance/delay)
- Time scaling
- Time reversal
- Dark UI
- Save graph feature

## Technologies Used
- Python
- NumPy
- Matplotlib
- Tkinter

## How to Run
1. Install required libraries:
   pip install numpy matplotlib
2. Run:
   python signals_advanced_gui.py
